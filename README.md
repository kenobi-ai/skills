# Kenobi Skills

AI agent skills for building with [Kenobi](https://kenobi.ai).

## Install

```bash
npx skills add kenobi-ai/skills
```

Or install a specific skill:

```bash
npx skills add kenobi-ai/skills --skill kenobi-pages
```

## Available Skills

| Skill | Description |
|---|---|
| `kenobi-pages` | Build personalized landing pages powered by Kenobi workflow content |

## What are skills?

Skills are portable instruction sets that teach AI coding assistants (Cursor, Claude Code, Codex, etc.) how to use tools effectively. They follow the open [Agent Skills](https://skills.sh) standard.
